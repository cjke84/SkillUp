# SkillUp Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a reusable `SkillUp` skill that packages and publishes skills to GitHub, Xiaping Skill, OpenClaw, and ClawHub without browser automation.

**Architecture:** A standard `SKILL.md` introduces a single shell entrypoint. A portable shell script coordinates validation, discovery, packaging, config loading, and platform adapters. Each platform gets an isolated adapter script to reduce coupling and ease future extension.

**Tech Stack:** Markdown, POSIX shell, `curl`, `zip`, `git`, optional `gh`
